# 11 - Assincronismo

## Conteúdo
- Callbacks
- Promises
- async / await
- Fetch API (exemplo para browser)

## O que aprender
Este módulo apresenta os principais padrões para lidar com operações assíncronas em JavaScript, como requisições HTTP e temporizadores.

## Como usar
3. Para testar `fetch`, use um navegador ou configure `node-fetch` no Node.

## Aprendizado
- Callbacks são funções executadas após uma operação terminar.
- Promises representam uma operação assíncrona futura.
- `async/await` simplifica o fluxo assíncrono.
- `fetch` é a API padrão para chamadas HTTP no navegador.

## Desafio
Implemente uma função assíncrona que aguarde uma promise e mesmo se houver erro, trate a exceção com `try/catch`.
