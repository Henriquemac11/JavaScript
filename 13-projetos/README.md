# 13 - Projetos práticos

## Conteúdo
- Calculadora
- To-do list
- Jogo de adivinhação
- Consumo de API

## O que aprender
Este módulo reúne projetos reais que aplicam conceitos de JavaScript em situações concretas e demonstram capacidade de construção prática.

## Como usar

## Aprendizado
- Estruturar funções para operações reutilizáveis.
- Gerenciar estado de uma lista de tarefas.
- Criar lógica de jogo simples.
- Consumir APIs externas com `fetch`.

## Desafio
- Expanda a calculadora para receber operações de um objeto.
- Implemente remoção de tarefas na lista.
- Ajuste o jogo para receber tentativas em vez de um chute fixo.
