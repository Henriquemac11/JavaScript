# JavaScript Practice

Este repositório é um curso prático de JavaScript do básico ao avançado, organizado em módulos sequenciais.

## Objetivo
- Documentar um aprendizado progressivo de JavaScript.
- Fornecer exemplos executáveis e didáticos.
- Servir como portfólio técnico no GitHub.

## Conteúdo do projeto
- 01-variaveis
- 02-comandos-basicos
- 03-operadores
- 04-condicionais
- 05-repeticao
- 06-arrays
- 07-matrizes
- 08-funcoes
- 09-poo
- 10-estruturas-avancadas
- 11-assincronismo
- 12-manipulacao-dom
- 13-projetos

Este projeto demonstra evolução clara em JavaScript, com foco em código limpo, boas práticas e estruturação profissional. Cada módulo inclui teoria, exemplos e desafios para reforçar o aprendizado.
