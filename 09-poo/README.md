# 09 - Programação Orientada a Objetos (POO)

## Conteúdo
- Classes
- Encapsulamento
- Herança
- Polimorfismo
- Abstração

## O que aprender
Este módulo explora os conceitos de orientação a objetos em JavaScript, usando classes e herança para modelar problemas de forma organizada.

## Como usar

## Aprendizado
- Use `class` e `constructor` para criar modelos de dados.
- Proteja propriedades com getters e setters.
- Estenda classes com `extends`.
- Reescreva métodos para criar comportamentos especializados.

## Desafio
Implemente uma classe `Usuario` com `nome`, `email` e método `apresentar()`.
