# 08 - Funções

## Conteúdo
- Funções tradicionais
- Arrow functions
- Parâmetros e retorno
- Funções de ordem superior (callbacks)

## O que aprender
Este módulo apresenta blocos reutilizáveis de código, incluindo como passar funções como argumentos e trabalhar com valores de retorno.

## Como usar

## Aprendizado
- Defina funções para isolar lógica.
- Use arrow functions para sintaxe concisa.
- Use callbacks para maior flexibilidade.

## Desafio
Crie uma função que calcule a média aritmética de um array de números.
