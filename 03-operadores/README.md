# 03 - Operadores

## Conteúdo

## O que aprender
Este módulo explora os operadores fundamentais de JavaScript, usados em cálculos e tomadas de decisão.

## Como usar

## Aprendizado
- `+`, `-`, `*`, `/`, `%`, `**`
- `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`
- `&&`, `||`, `!`

## Desafio
Crie uma validação que confirme se um usuário pode realizar uma compra com base em `saldo` e `precoCompra`. Use operadores lógicos para obter um resultado booleano.
