# 10 - Estruturas avançadas

## Conteúdo
- Destructuring
- Spread / rest
- Closures

## O que aprender
Este módulo introduz recursos modernos do JavaScript que tornam o código mais conciso e poderoso, especialmente para manipulação de objetos e funções.

## Como usar

## Aprendizado
- Extraia valores de objetos e arrays com destructuring.
- Combine arrays e objetos com spread.
- Crie funções que recebem quantidade variável de argumentos com rest.
- Mantenha estado privado com closures.

## Desafio
Combine dois arrays com spread e some os valores usando uma função que usa rest.
