# 07 - Matrizes

## Conteúdo
- Arrays bidimensionais
- Percorrer matrizes com loops aninhados
- Converter matrizes para objetos

## O que aprender
Este módulo expande o uso de arrays para tabelas de dados e mostra como trabalhar com coleção de linhas e colunas.

## Como usar

## Aprendizado
- Matrizes podem representar tabelas de dados.
- Use loops aninhados para percorrer cada célula.
- Converta estruturas em objetos para facilitar o uso.

## Desafio
Adicione uma nova linha à matriz e exiba o resultado atualizado.
