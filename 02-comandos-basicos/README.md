## O que aprender
Este módulo ensina como exibir informações no console, como trabalhar com variáveis de diferentes tipos e como converter valores de string para número.

## Como usar

## Aprendizado
- `console.log` é a forma básica de depuração e saída.
- JavaScript tem tipos primitivos como `string`, `number`, `boolean`, `null` e `undefined`.
- Use `Number()` para converter strings numéricas em valores numéricos.

## Desafio
Simule uma compra simples: defina `precoUnitario` e `quantidade`, depois imprima o total formatado.
