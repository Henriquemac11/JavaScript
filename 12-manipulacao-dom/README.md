# 12 - Manipulação do DOM

## Conteúdo
- Seleção de elementos com `querySelector`
- Eventos com `addEventListener`
- Alteração de conteúdo e estilos
- Criação de elementos dinâmicos

## O que aprender
Este módulo ensina a interagir com a interface do navegador usando JavaScript puro, manipulando o DOM para atualizar a página em tempo real.

## Como usar
1. Abra `12-manipulacao-dom/index.html` no navegador.
2. Digite um texto e clique em `Enviar mensagem`.
3. Use o botão `Adicionar item` para inserir itens na lista.

## Aprendizado
- `document.querySelector` encontra elementos.
- `addEventListener` escuta eventos do usuário.
- `textContent`, `style` e `createElement` alteram a página dinamicamente.

## Desafio
Adicione um novo elemento de lista em `#listaMensagens` para cada texto digitado.
