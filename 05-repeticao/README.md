# 05 - Repetição

## Conteúdo
- `for`
- `while`
- `do while`
- Iteração de arrays e cálculos acumulativos

## O que aprender
Este módulo mostra como repetir ações de forma controlada usando estruturas de loop e como construir resultados progressivos.

## Como usar

## Aprendizado
- `for` é ideal para repetições com contador.
- `while` repete enquanto a condição for verdadeira.
- `do while` executa o bloco pelo menos uma vez.

## Desafio
Crie um array com os primeiros 10 números pares usando um loop e exiba no console.
