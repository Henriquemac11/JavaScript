# 04 - Condicionais

## Conteúdo
- `if`, `else if`, `else`
- `switch`
- Tomada de decisão em JavaScript

## O que aprender
Este módulo ensina como fazer diferentes caminhos de execução com base em condições e como organizar regras de negócio de forma clara.

## Como usar

## Aprendizado
- Use `if` quando precisar avaliar condições simples.
- Use `else if` para múltiplas opções.
- Use `switch` para valores discretos.

## Desafio
Crie uma lógica que converta a idade em categorias: `infantil`, `jovem`, `adulto` e `idoso`.
