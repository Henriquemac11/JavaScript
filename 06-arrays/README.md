# 06 - Arrays

## Conteúdo
- Criação e acesso a arrays
- Métodos úteis: `push`, `filter`, `map`, `reduce`
- Manipulação de coleções de dados

## O que aprender
Este módulo ensina como trabalhar com listas de valores e como usar métodos funcionais para transformar dados de forma legível e eficiente.

## Como usar

## Aprendizado
- Arrays são estruturas ordenadas.
- Use `map` para transformar elementos.
- Use `filter` para selecionar elementos.
- Use `reduce` para calcular um valor agregado.

## Desafio
Crie um array de objetos com `nome` e `idade` e filtre os maiores de 18.
